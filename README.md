# FastAPI 샘플 프로젝트

FastAPI의 다양한 기능을 실험하고 테스트하기 위한 플레이그라운드 프로젝트입니다.

## 🚀 테스트 기능들

- 커스텀 에러 핸들링
- 템플릿 통합 (Jinja2)
- 인증 및 권한 관리
- 요청 유효성 검사
- 비동기 작업 처리
- 파일 업로드
- 데이터베이스 연동
- API 문서화
- 미들웨어 구현
- 웹소켓 지원
- 백그라운드 태스크
- 의존성 주입

## 🛠 기술 스택

- FastAPI
- Python 3.9+
- Uvicorn
- Jinja2
- SQLAlchemy (예정)
- Pydantic
- pytest (테스트용)

```bash
# 가상환경 생성 및 활성화
python -m venv .venv
source .venv/bin/activate  # Windows의 경우: .venv\Scripts\activate
